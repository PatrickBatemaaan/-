## Лабораторная работа №1 - Функции ввода/вывода, условный оператор
### Задача 1 
Пользователь вводит длины трех сторон треугольника. Программа проверяет может ли такой треугольник существовать, позже ищет площади по формуле Герона. 
### Среда разработки 
Язык программирования: Python.  
Среда разработки: PyCharm 2025. 
### Инструкция по работе с программой
Необходимо ввести длины трех сторон для дальнейшей работы программы с данными. Значение может быть не целым числом.
